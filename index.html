GIS Script

The purpose of this project was to extract data from a node in a KML file that was inaccessible by the GIS software ESRI. By using the Python libraries arcPy and pyKML I was able to extract the Polygon data and successfully export it to a Geodatabase for ESRI to interact with. This script also, automated the process of extracting the data and adding the geo fields to a database which made the script reusable for future KML file data extraction. The code for the project can be found here.

Flask Website

Flask is one of the most popular microframeworks and one of the most popular frameworks in Python and the one I chose to build a basic blog using it. Using Flask I built a RESTful API that was connected to a SQL database. The blog app is fully functional and consists of a Post, Login, Register, Logout, About, and Contact me pages. Initially the SQL database used was a SQLite database but, to make the blog app functional when I deployed it to Heroku (host site) I switched the database to a PostgreSQL database. A link to the blog can be found at the bottom of this webpage and link to the repository can be found here.
